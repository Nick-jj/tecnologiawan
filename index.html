--- WAN ---

--- Enlaces Dedicados ---

1Líneas Arrendadas (Leased Lines):

Conexiones físicas dedicadas proporcionadas por un proveedor de servicios, como T1, E1 o fibra óptica. Uso: Ideal para conexiones de red permanentes y estables entre oficinas o sucursales.

2Fibra Óptica:

Enlaces dedicados que utilizan cables de fibra óptica para transmitir datos a alta velocidad. Uso: Adecuado para necesidades de alta capacidad y grandes volúmenes de datos

3Circuitos Privados Virtuales (VPCs):

Conexiones dedicadas que se establecen a través de la red pública, pero segmentadas para uso exclusivo. Uso: Utilizado para conectar redes en la nube con la infraestructura local de una empresa.

Tipos de Enlaces Conmutados

Redes de Conmutación de Paquetes

Transmiten datos dividiendo la información en paquetes que viajan de forma independiente. Ejemplos: Frame Relay, X.25.

Redes de Conmutación de Circuitos

Establecen un camino físico y continuo a través de la red para la duración de la llamada o sesión. Ejemplos: Redes telefónicas tradicionales

Acceso a Internet

Conexiones temporales a la red pública de Internet, como las conexiones dial-up. Ejemplos: Línea telefónica convencional para conexión de módem.

--- Point to Point Protocol ---

Establecimiento

LCP (Link Control Protocol): Configura y gestiona el enlace punto a punto, estableciendo parámetros de comunicación y realizando la negociación de opciones.

Autenticación

PAP (Password Authentication Protocol): Un método simple de autenticación basado en contraseñas.

Configuración y Control

IPCP (IP Control Protocol): Negocia la configuración de parámetros IP y asigna direcciones IP para el enlace.

Compresión

CCP (Compression Control Protocol): Permite la negociación de métodos de compresión de datos para optimizar el uso del ancho de banda..
